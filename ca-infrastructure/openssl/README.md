# OpenSSL PKI Lab

Target scripts:

- Create Root CA.
- Create Issuing CA.
- Issue citizen, officer, TSP, TSA, and OCSP responder certificates.
- Revoke certificates and generate CRLs.
- Verify certificate chains.

This folder should contain configuration templates only. Generated keys, CSRs, certificates, serial files, CRLs, and OCSP databases should be ignored unless they are small deterministic test fixtures.
