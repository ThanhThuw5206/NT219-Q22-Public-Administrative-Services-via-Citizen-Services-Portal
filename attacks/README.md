# Attack Simulations

Required experiments:

- replay remote signing request;
- verify document after signer certificate revocation;
- tamper signed document;
- verify LTV after TSA/OCSP outage;
- demonstrate UI deception or digest mismatch in client-side signing.

Each experiment should include:

- script or repeatable steps;
- expected result;
- observed result;
- logs or screenshots for the final report.
